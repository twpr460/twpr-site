TWPR.link 介紹

這是一個 自動 DNS 隧道更新工具。

專為「家用網路或 VPS 沒有公網 IP」設計的小工具，能夠 一鍵開啟 ngrok TCP 隧道，並自動把你的自訂網域名稱指向到 ngrok 公開地址。

這有什麼用？

  • ✅ 自動啟動 ngrok TCP 隧道
  • ✅ 不只適用於 Minecraft 伺服器，還能用於任何 TCP 服務，像是：
    • 遊戲伺服器（Minecraft、Valheim、Terraria、Rust...）
    • 私人網站、API、Socket 服務
    • 任何需要在家用網路後端打洞、想用自己域名對外的服務
  • ✅ 你的朋友可以直接用你自訂的網域來連線
  • ✅ 不需要知道 ngrok 的複雜地址

主要解決的問題

  • 🏠 沒有固定公網 IP 的家用網路
  • 🏢 社區大樓 NAT 內網
  • 🌐 想用自己的專屬網域來分享連線地址

誰適合使用？

  • 🎮 私人伺服器經營者
  • 👥 小型遊戲社群
  • 🧑‍💻 教學或開發者想快速分享測試服務
  • 💰 不想買 VPS 又想用自己的網域名稱

專業 DNS 轉發 / IP 指向服務

想把自己的網域名稱解析到指定 IP？
或是伺服器 IP 不固定想要固定？
我們提供快速、專業的 DNS 轉發服務，讓你的網域隨時指向目標伺服器。
就算 IP 不同，只要配合我們的工具都可以固定。

  • ✅ 自訂網域 → 專屬 IP
  • ✅ 適用網站、遊戲伺服器、應用程式
  • ✅ 穩定解析，快速生效

服務內容

  • ✅ DNS A/AAAA/CNAME 記錄設定
  • ✅ IP 轉發與解析管理
  • ✅ 可協助 Cloudflare 等代管平台設定
  • ✅ 後續解析修改支援
